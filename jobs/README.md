# blogs

## Project setup
```
npm install
```

## Run the JSON Server
```
json-server --watch "./src/data/blogs.json" --port 3000
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
